# Java Basics 

## Description
This repository contains beginner-level Java programs created to understand core programming concepts.

## Topics Covered
- Variables and Data Types
- Operators
- Conditional Statements (if-else, switch)
- Loops (for, while, do-while)
- Arrays
- Methods
- Pattern Programs
- Basic Problem Solving

## Programs Included
- Hello World
- Even or Odd
- Prime Number
- Factorial
- Fibonacci Series
- Pattern Printing
- Array Operations

## Purpose
This repository is created to build strong fundamentals in Java programming.
